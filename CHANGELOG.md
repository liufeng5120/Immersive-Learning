# Changelog

所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2025-12-18

### Added
- AI 智能替换：使用 LLM 将网页中的词汇替换为目标语言
- 悬浮提示：鼠标悬停显示原文
- 单词详情：点击单词查看音标、释义、例句
- 5 档接触强度调节
- 独立设置页面：完整的配置管理界面
- 多 API 配置管理：支持添加、编辑、删除多个 API 配置
- 配置导入导出：便捷的配置备份和迁移
- 双向语言学习：支持母语→目标语言和目标语言→母语
- 页面语言自动检测：智能识别页面语言并调整翻译方向
- 黑名单/白名单双模式：灵活的网站过滤策略
- 显示样式选项：译文(原文)、原文(译文)、仅译文三种模式
- API 错误处理：连续失败自动关闭，避免不必要的请求
- 实时设置同步：修改设置立即生效无需刷新
- 智能缓存：容量提升至 2400 条，支持 LRU 淘汰策略
- 右键菜单快速操作：添加/移除黑白名单
- 学习统计：记录每日/累计接触词数
- GitHub Actions 自动发布：基于 Git tag 的自动化发布流程

### Changed
- 配色方案调整为紫色主题
- 弹窗界面布局重构，新增导航标签
- 优化词汇标记视觉效果
- 改进暗色模式支持
- 缓存策略从时间过期改为 LRU 算法

### Fixed
- 默认关闭扩展，避免未配置时的错误
